Background

In order to study different subatomic particles, particle physicists need to use particle accelerators, like the Large Hadron Collider. Opposing beams of protons are accelerated to almost the speed of light and are smashed against each other, resulting in a cone-shaped spray of subatomic particles called jets.

There are several kinds of jets - in particular we observe single charmed quarks (QCD_c), single bottom quarks (QCD_b), the Higgs Boson which decays into two bottom quarks (H_bb), two charmed quarks (QCD_cc), two bottom quarks (QCD_bb), and all other jets (QCD_other). These categories depend on the decay and spray of the particles, as seen on the image to the right.

Previously, we trained a model to classify the Higgs boson against all other jets. Now, we want to expand our classifier to all jets.

The Problem

Based on the data generated in a particle collision, can we then classify a jet as one of six different classes?

The Challenge

While the morphologies of jets can make some classes easier to pick out, similarity in structure can cause others to be more difficult.

On top of that, there is an extreme imbalance of jet representation in the dataset.

Methods

DNN

- Fully Connected Dense Neural Network
- Baseline
- Prone to overfitting

CNN

- 1 Dimensional Convolutional Neural Network
- Modified Deep Sets Approach


GNN

- Graph Neural Network
- Interaction network model
- Uses Edge block, Node block, Global block

ENN

- Equivariant Neural Network
- Similar to graph network
- Respects symmetry in physics
- Too complex to implement in time period

Initial Performance

Before adding weights, our models struggled with poor performance. Even so, we can tell that the Higgs boson jet has the best classification amongst all the jets.

Final Results

After adding more datasets and incorporating class weights into the model, we see a huge jump in improvement for our models. Once again, we notice that the Higgs boson jet has a far more impressive performance from all three models compared to the other jets.

Goals and Impact


- Ex plored deep learning multiclassification techniques to classify 6 different categories of particle jets

- Compared several possible baseline models for jet tagging

- Simplified  the process from creating individual classifiers for each jet to training one large model.

We make an already tedious task more efficient. This project can be used as a stepping stone for future projects in the intricate world of particle physics.

Difficulties

- GNNs showed no significant improvements in performance when incorporating class weights
- Limited resources to study ENNs left us unable to implement a working model

Future directions

- A quantitative comparison between six individual models or one big model
- Fully implemented ENN
